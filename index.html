EasyZoom jQuery image zoom plugin

EasyZoom is an elegant jQuery image zoom plugin based on the original work by Alen Grakalic. As well as adding lots of whitespace to the source code it has been extended with public methods, optimised and includes an optional extension to create an image gallery. View the source code

← You can see it in action just to the left, simply move your mouse over the kitten or click the thumbnails below to change the image.
